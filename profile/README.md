# NeuraDock

NeuraDock builds brain-computer interface development tools for researchers, developers, and makers working with brain signals.

Our first open development platform is **NeuraDock EEG Workstation**, a 7-channel dry-electrode EEG development kit that combines wearable EEG hardware, recording software, Python tools, example workflows, and an open-source EEG analysis agent.

## NeuraDock EEG Workstation

NeuraDock EEG Workstation is designed for developers and research teams who want to prototype, record, analyze, and build applications with EEG data.

It includes:

- A 7-channel dry-electrode EEG headset
- A compact EEG acquisition module
- USB and Bluetooth data streaming
- Python tools for EEG data reading and analysis
- Example workflows for EEG experiments and signal processing
- An open-source EEG analysis agent for natural-language EEG workflows
- Hardware interface and port specifications for third-party integration

## Open Source Scope

NeuraDock is preparing open resources for EEG developers and researchers, including:

- EEG data reading examples
- Python-based analysis workflows
- Tutorial notebooks
- Example EEG experiments
- Agent workflows and prompts
- Hardware interface and port specifications for third-party integration

For hardware, the current open-source scope focuses on **hardware interface and port specifications**. Full hardware schematics, PCB design files, and manufacturing files are not included in the current public release scope.
